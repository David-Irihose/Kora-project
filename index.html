Job Finder Site

A school project that uses the Remotive API to fetch real remote job listings.

How This Project Works

Three simple steps to use this app

Search

Type any job title or skill and hit search to see live listings.

Filter

Filter by remote work or company name to narrow down results.

Apply

Click "Apply" on any job card and go directly to the listing.
